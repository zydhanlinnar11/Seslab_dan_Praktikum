Tower of Hanoi 2.0

Time Limit1 seconds
Memory Limit32 MB

Berhubung sudah banyak soal susah di Praktikum Final ini, para Asdos memberikan 1 soal yang tentunya sudah kalian kenal, permainan Tower of Hanoi (Menara Hanoi). Nah, kita telah belajar bahwa Tower of Hanoi memiliki total langkah penyelesaian sebanyak 2N-1 dan dapat dilakukan secara rekursif untuk mencari langkah-langkah pemindahan yang optimal, jika terdapat N keping yang harus dipindahkan.

Nah, di soal ini, kami hanya meminta kalian untuk mencetak langkah ke-K dari solusi optimal permainan Tower of Hanoi, jika diketahui jumlah kepingnya sebanyak N.

Input Format

Baris pertama T menunjukkan banyak testcase.

T baris berikutnya masing-masing terdiri dari N dan K.

Output Format

Keluarkan T baris yang masing-masing terdiri dari P : A => C, keping ke-P, yang dipindahkan dari tiang A ke tiang C.

Sample Input

3
2 3
3 5
3 7

Sample Output

1 : B => C
1 : B => A
1 : A => C

Explanation

Optimal solusi untuk Tower of Hanoi yang terdiri dari 2 keping adalah:

1 : A => B

2 : A => C

1 : B => C

Sehingga output dari testcase pertama adalah:

1 : B => C
