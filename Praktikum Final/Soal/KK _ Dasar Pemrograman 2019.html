Kabel Kabel

Time Limit1 seconds
Memory Limit64 MB

Diberikan N segmen dari kabel dengan panjang L1, L2, ..., LNcm. Dibutuhkan pemotongan menjadi K bagian yang sama panjangnya atau dapat lebih panjang, yang diekspresikan dalam cm. Jika pemotongan menjadi K segmen dengan panjang 1cm pun tidak dapat dilakukan, maka jawabannya adalah 0.

Input Format

Pada baris pertama, terdapat N dan K. Kemudian N baris berikutnya adalah L1, L2, ..., LN.

Output Format

Panjang dari segmen tali yang dipotong.

Sample Input

4 11
802
743
457
539

Sample Output

200

Explanation

Kabel 1: 802/200 = 4 segmen

Kabel 2: 743/200 = 3 segmen

Kabel 3: 457/200 = 2 segmen

Kabel 4: 539/200 = 2 segmen

4+3+2+2 = 11 segmen

Constraints

  • 1 ≤ N ≤ 10000
  • 1 ≤ K ≤ 10000
  • 100 ≤ Li ≤ 10000000
