Ngakak Onlen

Time Limit1 seconds
Memory Limit8 MB

Hanji merupakan seseorang yang suka menganalisa tawa teman-temannya pada sosmed. Menurut dia, ada tawa yang benar-benar tulus, ada yang tidak. Hasil analisanya adalah sebagai berikut:

  • Huruf konsonan tidak berpengaruh
  • Jika tawa seseorang dibalik dan urutan huruf vokalnya sama seperti sebelum dibalik, maka tawa tersebut merupakan tawa yang tulus.

Bantu Hanji membuat program untuk menentukan apakah tawa teman-temannya tulus atau tidak.

Input Format

Satu baris tawa.

Output Format

Jika tawa tulus, tampilkan "ngakak beneran".

Jika tawa tidak tulus, tampilkan "pura-pura ngakak".

Sample Input 1

wkaowkoakwaokwoawk

Sample Output 1

ngakak beneran

Explanation 1

urutan huruf vokal: aooaaooa

urutan jika dibalik: aooaaooa

urutan huruf vokal == urutan huruf vokal jika dibalik

Sample Input 2

heuheuheuheu

Sample Output 2

pura-pura ngakak

Explanation 2

urutan huruf vokal: eueueueu

urutan jika dibalik: ueueueue

urutan huruf vokal != urutan huruf vokal jika dibalik

Constraints

  • Satu baris tawa terdiri atas maksimal 50 karakter.
  • Setiap karakter tawa dijamin merupakan huruf kecil.
  • Pada satu baris tawa terdapat setidaknya 1 huruf vokal.
