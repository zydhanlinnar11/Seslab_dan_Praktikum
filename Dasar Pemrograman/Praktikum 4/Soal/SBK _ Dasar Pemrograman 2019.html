Sibuk

Time Limit1 seconds
Memory Limit64 MB

Kali ini, karena TimTim sibuk dengan banyaknya dedlen (dan juga menyelesaikan TA) dan urusan lainnya, TimTim berencana memberikan soal yang mudah. Terdapat n query, dimana hanya terdapat 2 kemungkinan query yaitu,

1 x. Dimana menambahkan bilangan x ke dalam list.

2. Keluarkan bilangan terakhir dari list tersebut, keluarkan index minimum (index dimulai dari 0) dari bilangan minimum dari index awal sampai index terakhir dari list tersebut, keluarkan nilai minimum dari index awal sampai index terakhir, lalu keluarkan index terakhir dari list tersebut. Atau keluarkan "KOSONG!!" (Tanpa tanda petik) apabila list tersebut kosong. Untuk penjelasan lebih lanjut, lihat sample output.

Input Format

Baris pertama berisi 1 buah bilangan bulat n yang menyatakan banyaknya query.

n baris selanjutnya berisi query 1 ataupun 2.

Output Format

Keluarkan "KOSONG!!" (tanpa tanda petik) apabila list tersebut kosong, atau keluarkan 2 buah baris yaitu,

"Nilai sekarang : x" (tanpa tanda petik) dimana x menyatakan nilai sekarang

"Nilai minimum berada pada index ke-y dengan nilai z" (tanpa tanda petik) dimana z menyatakan nilai minimum dan y menyatakan index minimum dari nilai z

Sample Input

6
2
1 10
1 15
1 8
2
2

Sample Output

KOSONG!!
Nilai sekarang : 8
Nilai minimum berada pada index ke-2 dengan nilai 8
Nilai sekarang : 15
Nilai minimum berada pada index ke-0 dengan nilai 10

-

Constraints

  • 1 ≤ n ≤ 200000
  • -100000 ≤ x ≤ 100000
