Mus Lucky Cube

Time Limit10 seconds
Memory Limit512 MB

Leeta adalah seorang gadis yang gemar bermain teka-teki. Suatu hari, ia mendapatkan hadiah dari seseorang yang tidak ia kenal. Hadiah tersebut berupa sebuah kubus yang diberi nama Lucky Cube. Lucky Cube terdiri dari N * N * N buah kubus kecil. Si pemberi hadiah berkata bahwa Leeta akan menemukan suatu peti yang berisi sebuah surat di suatu kebun milik si pemberi hadiah tersebut jika ia dapat menyelesaikan teka-teki pada Lucky Cube tersebut. Karena rasa ingin tahunya yang tinggi ia pun menerima tantangan itu.

Permainannya seperti ini:

  1. Tiap-tiap kubus kecil pada Lucky Cube memiliki suatu bilangan bulat.
  2. Misalkan Lucky Cube tersebut berada pada sebuah sistem koordinat dimana ujung kiri bawah depan nya terletak pada 0, 0, 0 dan ujung kanan atas belakang nya terletak pada N - 1, N - 1, N - 1
  3. Bilngan yang terdapat di kubus kecil pada Lucky Cube dapat dinotasikan sebagai suatu himpunan <x, y, z> dimana bilangan x, y, dan z yang menyatakan koordinatnya.
  4. Terdapat 3 perintah pada Lucky Cube tersebut yaitu:
    • Perintah 1: menambah <x, y, z> sebanyak a
    • Perintah 2: menukar <x1, y1, z1> dengan <x2, y2, z2>
    • Perintah 3: Mengeluarkan jumlah semua kubus kecil pada Lucky Cube kecuali yang sejajar dengan <x, y, z>

Karena kerepotan, Leeta meminta bantuanmu untuk menyelesaikannya. Dapatkah kamu melakukannya?

Input Format

Baris pertama berisi sebuah bilangan bulat N yang menyatakan ukuran Lucky Cube sesuai deskripsi soal.

N ^ 3 baris selanjutnya berisi bilangan pada kubus-kubus kecil pada lucky cube dengan urutan input sebagai berikut

<0, 0, 0>

<0, 0, 1>

...

<0, 0, N - 1>

<0, 1, 0>

...

<0, N - 1, N - 1>

<1, 0, 0>

...

<N - 1, N - 1, N - 1>

Baris selanjutnya berisi sebuah bilangan bulat Q yang menyatakan banyaknya perintah yang akan dilakukan

Q baris selanjutnya berisi 3 macam perintah yang diawali dengan nomor perintah:

1 x y z a (perintah pertama)

2 x1, y1, z1, x2, y2, z2 (perintah kedua)

3 x y z (perintah ketiga)

Output Format

Bilangan bulat sebanyak perintah ketiga sesuai dengan deskripsi soal.
