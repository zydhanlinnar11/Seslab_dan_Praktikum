Lelang Sepatu

Time Limit1 seconds
Memory Limit32 MB

Drigo adalah seorang pengusaha sepatu sekaligus juga seorang programmer yang handal. Usaha sepatunya saat ini sedang berkembang pesat, dan ia kini butuh banyak sekali tempat untuk membangun toko di kotanya. Selain itu, Drigo juga mempunyai banyak koleksi sepatu (satu jenis sepatu) impor dan mahal dalam berbagai ukuran. Setiap koleksi sepatu mahalnya ditandai dengan suatu ID berupa string. Untuk mengetahui ukuran dari sepatu tersebut, tinggal dilihat saja digit-digit penyusunnya. Misalkan, ID nya adalah “serialxx12abc0”, maka ukuran sepatu tersebut adalah 120.

Drigo berencana untuk menambah modal membuka toko sepatu dengan melelang semua sepatu impornya. Masalahnya adalah Drigo tidak tahu berapa pasang sepatu yang dapat ia lelang sebab beberapa sepatu impornya telah rusak dan tidak bisa digunakan. Bantulah ia untuk menemukan solusi atas permasalahannya.

Input Format

Baris pertama pada input adalah N, yakni banyaknya sepatu impor layak pakai yang dimiliki Drigo. N baris berikutnya terdapat string (ID dari sepatu) dan satu buah karakter. Karakter ‘L’ menandakan untuk sepatu kiri dan ‘R’ menandakan sepatu kanan.

Output Format

Output terdiri dari satu bilangan yang menyatakan banyaknya pasang sepatu yang bisa ia lelang.

Sample Input

5
seri34 L
exe3n4 R
sepa1tu25 L
125sepatu R
iup233 L

Sample Output

2

Explanation

Terdapat sepasang sepatu berukuran 34, dan sepasang sepatu berukuran 125. Sepatu terakhir tidak mempunyai pasangan yang valid, sehingga hanya ada 2 sepatu yang bisa Drigo lelang.

Constraints

  • 1 ≤ panjang string ≤ 100
  • 1 ≤ N ≤ 100000
  • String hanya terdiri dari huruf lowercase dan digit
  • Ukuran sepatu tidak melebihi 210-1
