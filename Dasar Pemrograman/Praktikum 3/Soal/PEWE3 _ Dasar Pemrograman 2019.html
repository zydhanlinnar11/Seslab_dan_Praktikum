PW3

Time Limit1 seconds
Memory Limit8 MB

Setelah berhasil menyelesaikan misinya, N diberi hadiah oleh raja berupa sebungkus  coklat batangan. Coklat ini tidak biasa, satu potong coklat bisa membuat kenyang selama 10 tahun. N sangat senang dengan hadiah ini karena bisa dibagi-bagi ke adik2 nya. Masalahnya, bungkus coklat tersebut memiliki puzzle persamaan linier khusus dengan maksimal 3 variabel.  Tugasmu membantu N untuk menyelesaikan puzzlenya dengan mencari berapa banyak kemungkinan penyelesaiannya, mudah bukan?. 

 

Input Format

baris pertama banyaknya testcase n

baris kedua merupakan banyak koefisien, koefisien x y atau z juga, dan hasilnya h

 

Output Format

"Puzzle Salah, coba lagi!" jika tidak memiliki penyelesaian

"Puzzle Benar 4" kata puzzle benar diikuti banyaknya kemungkinan

keduanya tanpa petik

 

Sample Input 1

4
2 2 2 6
2 1 2 3
2 2 2 1
3 5 1 2 4

Sample Output 1

Puzzle Benar 4
Puzzle Benar 2
Puzzle Salah, coba lagi!
Puzzle Benar 3

Sample Input 1

3
2 3 1 4
3 1 2 3 4
3 3 3 3 1

Sample Output 1

Puzzle Benar 2
Puzzle Benar 4
Puzzle Salah, coba lagi!

Explanation

banyakvar = 2, pers = {2, 2}, hasil = 6, representasinya 2x + 2y = 6   jadi banyak kemungkinannya 4

{x=1, y=2}, {x=2, y=1}, {x=0, y=3}, {x=3, y=0}

 

banyakvar = 3, pers = {5, 1, 2}, hasil = 4, representasinya 5x + y + 2z = 4  jadi banyak kemungkinannya 3

{x=0, y=2, z=1}, {x=0, y=4, z=0}, {x=0, y=0, z=2}

 

dan x y z pasti bilangan bulat

 

Constraints

  • 1 ≤ n ≤ 10
