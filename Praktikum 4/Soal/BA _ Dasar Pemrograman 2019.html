Bank Alprofit

Time Limit1 seconds
Memory Limit8 MB

Kade adalah pemilik Bank Alprofit. Kade ingin membuat program untuk mengelola Bank Alprofit, namun ia tidak bisa ngoding. Kamu, sebagai programmer handal sepanjang masa, memutuskan untuk membantu Kade untuk membuatkan program tersebut. Program tersebut dapat membuatkan sejumlah p akun. Setiap akun yang dibuat memiliki ID, nama pemilik, dan saldo. Setiap akun memiliki saldo awal sebanyak 100. Program tersebut dapat melakukan penyetoran atau penarikan uang dengan jumlah uang J yang dikehendaki sang pemilik akun. Namun, ada syarat sebagai berikut:

  • Jika saldo saat ini lebih dari 1000, maka setiap akun tersebut melakukan penyetoran uang, setorannya ditambah dengan 20.
  • Jika jumlah uang yang ditarik melebihi saldo saat ini, maka seluruh saldo dari akun tersebut akan ditarik, sehingga saldo saat ini menjadi 0.

Input Format

Baris pertama berisi satu bilangan bulat p, menandakan jumlah akun yang akan dibuat.

Kemudian, p baris berikutnya berisi ID serta nama pemilik akun.

Selanjutnya, terdapat 3 pilihan input yang dapat dilakukan selama pengguna belum memilih untuk keluar dari program:

  • Jika input adalah 1, 2 input berikutnya adalah ID dan jumlah uang J yang akan disetorkan.
  • Jika input adalah 2, 2 input berikutnya adalah ID dan jumlah uang J yang akan ditarik.
  • Jika input adalah 0, maka keluar dari program.

Output Format

Pada akhir program, tampilkan informasi dari seluruh akun, dengan format sebagai berikut:

Akun #

[nama]

[saldo akhir]

#: urutan input

Saldo ditampilkan dengan 2 angka di belakang koma.

Sample Input

2
1 fandy
2 alam
1
1 500
1
2 2000
2
1 1000
0

Sample Output

Akun 1
fandy
0.00
Akun 2
alam
2120.00

Explanation

Saldo awal

1 -> Fandy -> 100

2 -> Alam -> 100

Input pertama,

Dilakukan penyetoran uang sebanyak 500 ke akun dengan ID 1 (Fandy), sehingga saldonya menjadi 600.

Input kedua,

Dilakukan penyetoran uang sebanyak 2000 ke akun dengan ID 2 (Alam), sehingga saldonya menjadi 2100. Karena saldonya sudah melebihi 1000, maka saldonya ditambah dengan 20, sehingga saldonya menjadi 2120.

Input ketiga,

Dilakukan penarikan uang sebanyak 1000 dari akun dengan ID 1 (Fandy). Karena saldonya kurang dari jumlah uang yang akan ditarik, maka seluruh saldo akan ditarik, sehingga saldonya menjadi 0.

Input keempat,

Program selesai.

Constraints

  • 1 ≤ p ≤ 100
  • 0 ≤ J ≤ 100000
  • Setiap ID dijamin unik.
